# Sensr DataBridge

Data integration solution for private capital sponsors.

**Tagline:** "Bridge the Data Gap -- Your Data Anywhere You Need It"

## What It Does

DataBridge lets private capital sponsors access their Gen II fund administration data -- investor cash flows, investment cash flows, portfolio data, and accounting -- within their preferred platforms. It bridges the gap between Gen II's data and whatever in-house or third-party tools the sponsor uses.

## Key Features

- **Seamless Integration** -- Connects Gen II fund admin data with both in-house and third-party platforms (portfolio management systems, CRM, reporting dashboards, data warehouses).
- **Customizable Data Feeds** -- Proprietary "DIY" on-screen extract designer. Users tailor data feeds to specific needs without external support.
- **Flexible Access Options** -- Two delivery methods:
  - Scheduled flat file delivery (for batch processing)
  - Direct API integration (for real-time consumption)
